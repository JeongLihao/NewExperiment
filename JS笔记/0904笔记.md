# JavaScript书写位置
+ 行内样式表
+ 内部样式表
+ 外部样式表

1. 内部
```js
 <script>
    alert('你好JS')
</script>
</body>
```
2. 外部
```js
<script src="地址"></script>
```
3. 内联
```js
 <script>
    alert('你好JS')
    document.write('JavaScript我来了！')
    console.log('魔法')
</script>
```
